  <name>Task 1: Create RouteMap component with MapLibre GL</name>
  <files>src/components/RouteMap.tsx, src/components/RouteMap.css</files>
  <action>
    1. Create `src/components/RouteMap.tsx`:
       - Accept props: `{ records: ActivityRecord[] }` (import type from `../types/activity.ts`)
       - Import `{ Map, Source, Layer }` from `@vis.gl/react-maplibre`
       - Extract GPS coordinates: filter records where BOTH positionLat AND positionLong are non-null, map to `[positionLong, positionLat]` tuples (GeoJSON is [lng, lat] order -- see RESEARCH.md Pitfall 6)
       - If fewer than 2 valid coordinates, return null (nothing to render)
       - Compute bounding box from coordinates for auto-zoom: find min/max lng and lat
       - Build GeoJSON Feature with LineString geometry from coordinates
       - Render `<Map>` with:
         - `initialViewState={{ bounds: [[minLng, minLat], [maxLng, maxLat]], fitBoundsOptions: { padding: 30 } }}`
         - `mapStyle="https://basemaps.cartocdn.com/gl/dark-matter-gl-style/style.json"`
         - `style={{ width: "100%", height: "100%" }}`
         - `attributionControl={false}` (clean HUD look)
       - Inside Map: `<Source id="route" type="geojson" data={geojson}>` with `<Layer id="route-line" type="line" paint={{ "line-color": "#00ffaa", "line-width": 2.5, "line-opacity": 0.85 }} />`
       - Wrap Map in `<div className="route-map">`

    2. Create `src/components/RouteMap.css`:
       - `.route-map`: height 100%, width 100%, border-radius 8px, overflow hidden, background rgba(10,14,20,0.6), border 1px solid rgba(255,255,255,0.06)
       - Override MapLibre's canvas border-radius to match container: `.route-map .maplibregl-canvas { border-radius: 8px; }`

    3. Verify: `bunx tsc --noEmit`
  </action>
