<task type="auto">
  <name>Task 1: Create Dashboard container with CSS grid layout and conditional panels</name>
  <files>src/components/Dashboard.tsx, src/components/Dashboard.css</files>
  <action>
    1. Create `src/components/Dashboard.tsx`:
       - Import useActivityStore from store, detectChannels from data-presence
       - Import StatsPanel, all 4 chart components, RouteMap, LapTable
       - Import Dashboard.css
       - Get activity from store: `useActivityStore((s) => s.activity)`
       - If activity is null, return null (should not render without data)
       - Compute `channels = detectChannels(activity.records)`
       - Compute `startTime = activity.records[0]?.timestamp.getTime() ?? 0`
       - Render structure:
         ```
         <div className="dashboard">
           <StatsPanel summary={activity.summary} />
           <div className="dashboard__charts">
             <div className="chart-panel chart-panel--wide">
               <PaceChart records={activity.records} startTime={startTime} />
             </div>
             {channels.hasHeartRate && (
               <div className="chart-panel chart-panel--wide">
                 <HeartRateChart records={activity.records} startTime={startTime} />
               </div>
             )}
             {channels.hasAltitude && (
               <div className="chart-panel chart-panel--half">
                 <ElevationChart records={activity.records} startTime={startTime} />
               </div>
             )}
             {channels.hasCadence && (
               <div className="chart-panel chart-panel--half">
                 <CadenceChart records={activity.records} startTime={startTime} />
               </div>
             )}
           </div>
           <div className="dashboard__bottom">
             {channels.hasGps && (
               <div className="dashboard__map-slot">
                 <RouteMap records={activity.records} />
               </div>
             )}
             {activity.laps.length > 0 && (
               <div className="dashboard__table-slot">
                 <LapTable laps={activity.laps} />
               </div>
             )}
           </div>
         </div>
         ```
       - IMPORTANT: When both bottom panels (map + laps) are absent, the bottom row should not render. When only one is present, it spans full width.

    2. Create `src/components/Dashboard.css`:
       - `.dashboard`: max-width 1400px, margin 0 auto, padding 4.5rem 1.5rem 2rem (same top padding as skeleton for header clearance)
       - `.dashboard__charts`: display grid, grid-template-columns 1fr 1fr, gap 0.75rem, margin-top 1rem, margin-bottom 1rem
       - Chart panel heights: `.chart-panel--wide` height 160px (pace and HR span 2 cols), `.chart-panel--half` height 140px
       - `.dashboard__bottom`: display grid, grid-template-columns 1fr 1fr, gap 0.75rem
       - `.dashboard__map-slot`: height 240px
       - `.dashboard__table-slot`: height 240px, overflow-y auto
       - When bottom has only one child, it should auto-span: `.dashboard__bottom > :only-child { grid-column: 1 / -1; }`
       - Responsive: at max-width 768px, charts grid goes to 1 column, bottom grid goes to 1 column

    3. Verify: `bunx tsc --noEmit`
  </action>
  <verify>
    <automated>bunx tsc --noEmit</automated>
  </verify>
